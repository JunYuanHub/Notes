常见状态码：  
```  
101 :协议切换，socket  

200 ok  
201 创建成功，新的资源根据请求建立  
202 Accepted，新的资源请求接受，但还没创建
206 Partial Content，实现断点续传，请求头需要包含range

301 永久重定向（可缓存，下次在请求直接访问该新地址）
302 临时重定向（除非设置Cache-Control或Expires，否则还会重新访问原地址）  
304 Not Modified。请求资源未改，缓存资源可以继续使用

400 语义、参数错误，服务器无法理解
401 Unauthorized。请求需要验证
403 请求被拒绝
404 找不到请求资源  

500 执行请求时内部错误
502 网关、代理服务器错误。上游服务器收到无效请求。
503 过载或停机维护。暂时的

```
