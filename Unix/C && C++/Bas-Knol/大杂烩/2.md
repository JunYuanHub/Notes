# 字符串与标准输入输出  
## 字符串  
字符串在内存存储中以\0结尾，故字符串常量"x"与字符常量'x'不同，前者在内存中实际上是字符'x'和空字符'\0'  
1. scanf("%f",&a)读取时遇到空格符便结束了。  
2. strlen()不包含空字符，sizeof()会  
3. \# define NAME value:编译时替换，编译程序时所有的NAME会被替换为value，这样定义的常量也成为了明示常量。  
    C90新增const
