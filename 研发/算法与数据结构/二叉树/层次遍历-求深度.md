# 一、层次遍历求深度  
借助队列，每次出队完一层的所有节点后深度加一 => 关键在于如何确定一层的结点全部出完 => 记下length，出队length个元素
```  
/* function TreeNode(x) {
    this.val = x;
    this.left = null;
    this.right = null;
} */
function TreeDepth(pRoot)
{
    // write code here
    if(!pRoot){ return 0}
    
    let a=[pRoot],
        dep=0
    while(a.length){
        let size = a.length
        while(size){
            let tem = a.shift()
            tem.left?a.push(tem.left):null
            tem.right?a.push(tem.right):null
            size--
        }
        dep++
    }
    return dep
}
```
