# Linux常用命令手册
### 系统操作命令  
通用命令格式：命令 参数 目录  
1. 进程与服务相关   
    ```
    shutdown -halt/-h    t   :在t分钟后关机，t为0或者now时立即关机
    shutdown -cancle/-c      :取消关机（需要权限足够）
    shutdown -reboot/-r  t   :在t分钟后重启
    reboot                   :直接重启
    halt/poweroff            :直接关机
    
    ps          :查看进程
    ps -ef      :查看所有详细的进程信息
    kill        :删除指定进程
    kill pid    :删除指定pid的进程
    kill -9 pid :强制删除pid的进程
    
    free -h     :查看内存命令.+h以友好方式查看
    lscpu       :查看CPU信息
    w           :查看负载信息、系统用户登录信息
    
    systemctl                   :管理服务程序的运行状态
    systemctl start       	    :启动服务
    systemctl stop 	      	    :停止服务
    systemctl restart    	    :重启服务
    systemctl status      	    :查看服务详细的运行状态
    systemctl disable     	    :让服务开机不要运行
    systemctl enable      	    :让服务开机运行
    systemctl is-active  	    :检查确认服务是否运行
    systemctl is-enabled 	    :检查确认服务是否开机运行
      
    netstat -lntup|grep 端口号   :查询端口号服务情况
    ```  
2. 目录/文件操作  
    ```
    1.ls             :检查文件是否存在，列出目录中内容。
        -d           :检查目录信息（不会列出目录中的内容）
        -l           :显示文件属性
        -a           :查看包括隐藏文件在内
        -lt          :将目录中的信息按照时间进行排序显示
    2.mkdir          :创建目录，如果已存在会报错
        -p/parent    :创建多级目录，目录已存在不会报错
    3.cd dir/-/~     :进入dir，加 - 进入上一个操作的目录，加 ~ 进入家目录
    4.pwd            :显示当前路径。(print working dir)
    5.touch          :创建空文件
    6.which          :查看命令所在目录(从环境变量中寻找)
    7.whereis        :显示命令路径以及命令相关的帮助手册路径
    
    7.cat file           :查看文件
    8.tail -x file       :查看倒数x行，默认10，【-f】追踪查看
      head -x file       :查看前x行，默认10
    9.cp file dir        :将file复制到dir中
         -r              :加r可以复制目录，r:recursion递归  
         -f              :遇到许多重复文件覆盖时，跳过确认
    10.rm file           :删除file
         -r              :删除目录
         -f              :强制删除，不询问
    11.mv file dir        :剪切，可用于文件重命名，即dir=>new filename  
    12.file xxx          ：查看文件属性
    ``` 
3. 文本编辑 
   1. 打开文件进行编辑：```vi/vim filename```  
      (1)按`i`进入编辑模式，编辑内容  
      (2)编辑模式中：dd删除一行，2dd删除两行，u撤回，#注释，shift+g移动到末尾，gg首部）  
      (3)按`esc`退出编辑模式，再输入操作：:w/:q/:wq，分别对应写入保存，退出，保存并退出  
   2. 不打开文件编辑  
      ```
      echo sth => 打印sth
      echo content > file  :将content写入file,覆盖之前的内容，file不存在会自动创建
      echo content >> file :将content追加写入file,不会覆盖，在最后一行追加
      >file可直接清空文件
      ```   
4. 用户与网络相关  
    ```  
    useradd       :创建用户(用户管理)
    passwd        :设置密码。加用户名指定修改该用户名密码，否则修改当前用户密码
    su            :切换用户命令
    whoami        :确认用户身份
    
    hostname      :查看用户名
      -i          :查看ip地址  
    dig 域名       :根据域名查找IP地址（需要yum安装）
    ```  
5. 软件程序相关  
    ```
    yum install -y xx               直接安装软件
    yum remove xx                   卸载软件，但只会卸载本身，卸载依赖建议用rpm
    yum groupinstall -y 包组         直接安装软件包组
    yum repolist                    查看yum源信息
    yum list                        查看可安装/已安装的软件
    yum grouplist                   查看可安装/已安装的软件包组
    yum --help                      显示命令的参数帮助信息
    yum provides 命令                获取命令属于哪个软件包
    
    rpm                 :管理软件程序包
        -qa 软件名称     :查看软件包是否安装成功
        -ql 软件名称     :查看软件包内容
        -qf 文件绝对路径  :查看命令文件属于哪个软件包  
        -ivh 软件.rpm    :手动安装软件包（无法解决软件依赖关系）
    ```
6. 其他命令  
    ```
    1.tr 'old' 'new'        :替换sth中的old为new，通常和管道一起用
      -c                    :反向替换，将不属于old的替换为new
      -d                    :删除old
      -s                    :将连续重复的字符变成单个
      
    2.xargs               :有些命令不接受stdin，无法使用管道，此时用他转化为命令行参数
                           将输入分隔为一个个参数并输出，默认以换行符和空格作为分隔；无法使用别名;单独使用同xargs echo
       -d                 :更改分隔符
       -L                 :当需要指定多个命令行参数时，指定多少行作为一个命令行参数。
       -nx                :指定每次将多少项，作为命令行参数。
       -p                 :打印出要执行的命令，询问用户是否要执行
       -t                 :打印出最终要执行的命令，然后直接执行，不需要用户确认。
       -i {}              :当后面的命令需要指定多个参数时，为避免参数混淆（如cp命令将目录与待复制的文件位置搞反），可以用{}指定输入的位置
       
    3.wc            :统计文件的行数、单词数、字符数
       -l           :统计行数
       -w           :统计单词数
       -c           :统计字符数  
    
    4.column -t     :创建表格形式显示数据
    ```
7. 帮助命令`man 命令 ：列举命令信息`  
### 快捷方式  
1. ctrl+c 终止操作  
2. ctrl+l 清屏操作  
3. ctrl+d 注销用户
4. ctrl+a 光标跳到输入首部
   ctrl+e 跳到输入尾部
   ctrl+左右 按单词跳
   ctrl+u 将光标位置到行首内容剪切
   ctrl+k 将光标位置到行末内容剪切
   ctrl+y 粘贴剪切的内容
5. tab    补全，补全目录或文件名称或命令
### 符号表示  
```  
~                          家目录符号
..                         上一级目录
.                          当前目录
>                          标准输出重定向符号
>>                         标准输出追加重定向符号
2>                         错误输出重定向符号
2>>                        错误输出追加重定向符号
<                          标准输入重定向符号
<<                         标准输入追加重定向符号


&&                         代表前一个命令执行成功后，再执行后面的命令
;                          代表前一个命令执行之后,再执行后面的命令
#                          1.代表将配置文件信息进行注释2.在命令提示符中表示超级管理员身份
$                          1.用于加载读取变量信息2.表示一行的结尾3.在命令提示符中表示普通用户身份
!                          强制
''                         不对内容做任何处理
""                         会对内容特殊信息做解析
``反引号==$()               将反引号中命令执行结果交给引号外面的命令进行处理
| 管道符号                  将前一个命令的结果交给管道后面命令进行处理
{} 序列符号(通配符)          通配符号,显示数字或字母的序列信息   
```
